# Lua de Mel Compartilhada — Daniel & Stefhane

Landing page convite para a lua de mel compartilhada em Barreirinhas — MA.

## Arquivo principal

O site funciona apenas com o arquivo:

```text
index.html
```

As imagens já estão embutidas dentro do HTML, então não é necessário subir pasta `assets`.

## Deploy no Vercel

No Vercel, importe este repositório do GitHub e use:

```text
Framework Preset: Other
Build Command: vazio
Output Directory: .
```
